# Module 3: SM-2 算法 — 面试复习调度

## Teaching Arc
- **Metaphor:** 肌肉记忆——太频繁练会累，太久不练会忘。SM-2 就是那个算出"最佳复习时间"的算法。
- **Opening hook:** 答对了一道题，下次还出吗？SM-2 说：看情况——对了拉长间隔，错了一天后重考。
- **Key insight:** SM-2 是记忆研究领域最经典的算法之一（1987年提出），它用两个数字：间隔天数和难度系数，动态算出下次复习时间。
- **"Why should I care?":** 知道这个算法，能让你要求 AI 工具"在对的时机提醒我复习"，而不是固定每天刷题。

## Code Snippets (pre-extracted)

File: backend/spaced_repetition.py (SM-2 核心算法)
```python
def sm2_update(sr_state: dict, score_0_10: float) -> dict:
    quality = min(5, int(score_0_10 / 2))  # 0-10分 → SM-2质量0-5
    ef = sr_state.get("ease_factor", 2.5)    # 难度系数，最小1.3
    reps = sr_state.get("repetitions", 0)

    if quality >= 3:  # 通过
        if reps == 0: interval = 1
        elif reps == 1: interval = 3
        else: interval = int(sr_state.get("interval_days", 1) * ef)
        reps += 1
    else:  # 失败，重置
        interval = 1
        reps = 0

    ef = max(1.3, ef + (0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02)))
    return {
        "interval_days": interval,
        "ease_factor": round(ef, 2),
        "repetitions": reps,
        "next_review": (date.today() + timedelta(days=interval)).isoformat(),
        "last_score": score_0_10,
    }
```
